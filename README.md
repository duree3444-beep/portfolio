# 🚀 MSA 아키텍처 지향 개발자 포트폴리오 (이규찬)

이 프로젝트는 기능 구현을 넘어 **서비스 구조와 도메인 분리**를 고민하는 개발자 이규찬의 가치관과 기술적 지향점을 담은 포트폴리오 웹사이트입니다.

## 🌟 핵심 컨셉: "구조가 기능을 정의한다"
단순한 화면 구성을 넘어, 서비스 간의 책임(Responsibility)을 명확히 나누고 데이터 흐름을 기반으로 확장 가능한 시스템을 설계하는 방식을 시각화하여 표현했습니다.

## 🛠️ Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS (Lucide Icons)
- **Animations**: Framer Motion
- **Architecture**: Domain-Driven Design (Concept), MSA Concept Visualization
- **Backend (Interests)**: Spring Boot, FastAPI, JWT Auth

## ✨ 주요 기능 및 특징
1. **아키텍처 다이어그램**: 서비스 간의 데이터 흐름과 도메인 분리(Domain Separation)를 시각적으로 표현하여 설계 철학을 강조합니다.
2. **반응형 디자인**: 다양한 디바이스(Desktop, Tablet, Mobile)에서 최적화된 사용자 경험을 제공합니다.
3. **핵심 역량 요약**: MSA, API 설계, 도메인 분리 등 실질적인 시스템 설계 철학을 직관적으로 소개합니다.
4. **프로젝트 심층 분석**: '하루 시그널' 프로젝트의 설계 배경, 문제 해결 과정, 구조적 이점을 상세히 기술합니다.

## 🚀 시작하기
이 프로젝트를 로컬에서 실행하는 방법은 다음과 같습니다.

### 설치
```bash
npm install
```

### 개발 모드 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```
